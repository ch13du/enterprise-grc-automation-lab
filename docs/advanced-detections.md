## Advanced Detection Engineering

This phase focuses on identifying suspicious behavioral patterns using KQL in Microsoft Sentinel.

### 1. Privilege Escalation Detection
Detects role assignment activity that may indicate unauthorized access elevation.

### 2. Activity Burst Detection
Detects unusually high number of operations performed by a single user within a short time window.

### 3. High-Risk Operations Detection
Monitors sensitive actions such as resource deletion and modification.

### Outcome
These detections simulate real-world SOC monitoring scenarios and demonstrate the ability to identify suspicious activity, create alert rules, and prepare for automated response.
